## 🔥 앱 버전

### 버전 설정

사용 가능한 두 버전 설정(versionCode, versionName)의 값을 모두 정의한다

<br>

### versionCode

- 양의 정수이며, 내부 버전 번호로 사용된다
- 이 번호는 한 버전이 다른 버전보다 최신인지를 판단하는 데 도움이 되며, 번호가 높을수록 최신 버전임을 나타낸다
- 사용자에게 표시되는 버전 번호가 아니다
