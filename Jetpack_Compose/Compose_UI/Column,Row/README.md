## 🔥 Column, Row

### LazyColumn

화면에 보여지는 컴포저블만을 표시하는 scrollable한 Column이다

LazyColumn은 화면에 보여지는 컴포저블만을 표시하기 때문에 화면 로딩 시간을 최적화 시킬 수 있어,

많은 아이템이 표시되어야 할 경우 필수적으로 사용되어야 하는 레이아웃이다
