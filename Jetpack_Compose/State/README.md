## 🔥 State

### State 정의

state(상태)는 시간이 지남에 따라 변할 수 있는 값이다

<br>

### 핵심 용어

- Composition

  : Jetpack Compose가 컴포저블을 실행할 때 빌드한 UI에 관한 설명이다

- Recomposition

  : 데이터가 변경될 때 컴포지션을 업데이트하기 위해 컴포저블을 다시 실행하는 것을 말한다
