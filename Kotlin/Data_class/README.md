## 🔥 Data class

### Data class

> 데이터 클래스는 주로 데이터를 저장하는데 사용된다

<br>

### Data class 특징

Data class는 toString(), hashCode(), equals(), copy() 메소드를 자동으로 정의하여 주기 때문에 다음과 같이 간단하게 클래스를 만들수 있게 된다
