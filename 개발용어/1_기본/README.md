## 🔥 개발 용어

#### 서버 / 클라이언트

- 서버 : 데이터나 기능을 제공하는 주체
- 클라이언트 : 서버에 요청하고 결과를 사용하는 주체 (모바일 앱, 웹)

#### 런타임 / 컴파일 타임

- 컴파일 타임 : 소스 코드를 실행 파일로 변환하는 시점
- 런타임 : 프로그램이 실제로 실행되는 시점
- 컴파일 에러는 실행 불가, 런타임 에러는 실행 중 발생

#### 프레임워크 / 라이브러리 / 플러그인

- 프레임워크 : 전체 구조 제공, 개발자가 틀 안에서 작업 (Android SDK)
- 라이브러리 : 필요한 기능만 가져다 사용 (Retrofit)
- 플러그인 : 기존 도구에 기능 추가 (Gradle Plugin)

#### 동기, 비동기

- 동기 : 순차 실행, 결과 대기
- 비동기 : 결과 기다리지 않고 다음 작업 실행
- 네트워크, IO 작업은 비동기 필수

#### 스레드, 코루틴

- 스레드 : OS 단위 실행 흐름, 비용 큼
- 코루틴 : 언어 수준의 경량 비동기 처리
- 코루틴은 스레드 위에서 동작

#### 메인 스레드

- UI를 그리는 스레드
- 블로킹 시 ANR 발생
- UI 작업만 담당해야함

#### 버그 / 에러 / 예외

- 버그 : 논리적 잘못된 코드
- 에러 : 컴파일 타임 문제, 실행 불가
- 예외 : 런타임 문제, 처리 가능 (try catch)

#### 디버깅

- 코드 실행 흐름을 추적해 문제를 찾는 과정
- 브레이크포인트, 로그 활용

#### 로깅

- 프로그램 실행 중 상태 기록
- 디버깅, 장애 분석, 사용자 이슈 파악에 사용

#### 메모리 관리

- 가비지 컬렉션 : 사용하지 않는 객체 자동 해제
- 메모리 릭 : 참조가 남아 메모리가 해제되지 않는 상태

#### 오버헤드

- 기능 추가로 발생하는 추가 비용
- 성능, 메모리, 네트워크 비용 포함

#### 난독화

- 코드 구조를 알아보기 어렵게 변환
- 리버스 엔지니어링 방지
- Android 에서는 Proguard / R8 사용

#### 단일 모듈 / 멀티 모듈

- 단일 모듈 : 구조 단순, 초기 개발 빠름
- 멀티 모듈 : 빌드 속도 개선, 역할 분리
- 규모 커질수록 멀티 모듈 유리

#### 보일러플레이트 코드

- 반복적이고 의미 없는 코드
- 유지보수 비용 증가
- 자동 생성이나 공통 모듈로 줄이는 것이 목표

#### 리팩토링

- 기능은 그대로 유지하며 구조 개선
- 가독성, 확장성, 유지보수성 향상 목적

#### REST API

- HTTP 기반 자원 중심 통신 방식

- GET / POST / PUT / DELETE
- 상태 비저장(stateless)

#### JSON

- 텍스트 기반 데이터 교환 포맷
- 가볍고 사람이 읽기 쉬움
- 모바일 ↔ 서버 표준

#### 세션 / 쿠키

- 세션 : 서버에 사용자 상태 저장
- 쿠키 : 클라이언트에 상태 저장
- 모바일에서는 주로 토큰 방식 사용

#### 데이터베이스 / DBMS

- 데이터베이스 : 구조화된 데이터 저장소
- DBMS : 데이터베이스를 관리하는 시스템
  - MySQL
  - Room

#### 관계형 데이터베이스

- 테이블 기반 데이터 구조
- 행(Row)과 열(Column)로 구성
- SQL 사용

#### 쿼리

- 데이터베이스에 보내는 명령
- SELECT(조회) / INSERT(삽입) / UPDATE(수정) / DELETE(삭제)

#### Git

- 분산 버전 관리 시스템
- 코드 변경 이력 관리
- 협업의 필수 도구

#### Pull Request (PR)

- 코드 변경 사항 리뷰 요청
- 협업 품질 관리 핵심 프로세스

#### 코드 리뷰

- 로직, 버그, 설계 검증
- 팀 컨벤션 공유
- 지식 전파 수단

#### CI / CD

- CI : 코드 통합 시 자동 빌드/테스트
- CD : 자동 배포 또는 배포 준비
- 품질 안정성과 개발 속도 향상

#### API (Application Programming Interface)

- 프로그램 간 상호작용 규칙
- 기능 사용을 위한 인터페이스

#### SDK (Software Development Kit)

- 개발에 필요한 도구, 라이브러리 묶음
- 플랫폼별 제공

#### IDE (Integrated Development Environment)

- 통합 개발 환경 (Android Studio)
- 코드 작성, 빌드, 디버깅 도구
