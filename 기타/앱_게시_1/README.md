## 11. 스토어 등록정보 설정

### 필수로 입력해야 하는 항목

- 앱 이름
- 간단한 설명
- 자세한 설명
- 앱 아이콘 (최대 1MB, 512x512 px)
- 그래픽 이미지 (최대 15MB, 1024x500 px)
- 휴대전화 스크린샷 (2장 이상, 크기는 1080x1920 px 추천)
- 7인치, 10인치 태블릿 스크린샷 (1장 이상, 크기는 1920x1080 px 추천)
